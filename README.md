# Todo App

A simple Todo App built using HTML, CSS, and JavaScript.

## Features

- Add Tasks
- Delete Tasks
- Mark Tasks Complete
- Local Storage Support

## Technologies Used

- HTML
- CSS
- JavaScript

## Future Improvements

- Dark Mode
- Task Priorities
- Search Functionality
